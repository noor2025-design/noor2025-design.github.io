# Project Overview
This page serves as my web design portfolio landing page, housing all of the 
projects I built across the MPS-CD program at Parsons School of Design. It gives 
users a single place to access all six of my web design projects completed during 
my time in Typography & Interaction.
## Concept / Process
The design of this index page was intentional in creating cohesion across 
all of my touchpoints, I used the same color palette as my product design 
portfolio as part of my personal brand. The editorial aesthetic carries 
through everything so that whether someone finds my work through this page 
or elsewhere, it feels consistent and recognizable as mine.

The page is structured around a flow with the loading animation 
as the focal point and first thing a user experiences, before they learn 
more about me through the about section, and then have the opportunity to 
explore each project individually.

## Design System
The color palette is drawn directly from my personal brand which is a bold yellow 
as the dominant accent, paired with a muted beige and chestnut brown to 
ground the editorial aesthetic.
For typography, Fahkwang is used for headings and Azeret Mono for body copy, creating an editorial contrast between display and technical type.
  
The page guides the user through a clear experience, starting with my name 
and role, moving into a brief about me introduction, and then laying out 
my six projects with simple animations that bring each card into view.
